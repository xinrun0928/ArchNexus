## 核心概念

**本地锁 vs 分布式锁**：

- 本地锁：synchronized、ReentrantLock
- 分布式锁：Redis 锁、Zookeeper 锁

**互斥锁 vs 读写锁**：

- 互斥锁：所有操作互斥
- 读写锁：读读不互斥，读写互斥

**自旋锁 vs 阻塞锁**：

- 自旋锁：不放弃 CPU，循环等待
- 阻塞锁：挂起线程

**可重入锁**：

- 同一线程可以多次获取
- synchronized 是可重入的

**公平锁 vs 非公平锁**：

- 公平锁：FIFO 顺序
- 非公平锁：可能插队，性能更高

:::tip 学习建议

理解每种锁的适用场景比记住实现细节更重要。选择错误的锁类型比不使用锁更危险。
:::
