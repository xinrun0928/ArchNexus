# 会话管理

会话管理是保护用户身份不被冒用的关键机制。

## 核心议题

Session 认证面临的威胁包括会话劫持、会话固定、CSRF 等，每种攻击都有对应的防护手段。

会话存储策略决定了系统的扩展能力，是分布式系统必须解决的问题。

## 内容导览

| 文章 | 核心内容 |
|---|---|
| [会话劫持](./hijacking) | Session ID 窃取，防护措施 |
| [会话固定攻击](./fixation) | 攻击者预设 Session ID，登录后重放 |
| [CSRF 防护](./csrf) | 跨站请求伪造，Token 防护 |

## 攻击与防护

```
┌─────────────────────────────────────────────────────────────────┐
│                    会话安全威胁                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  会话劫持：攻击者窃取 Session ID                               │
│  ├── 防护：HTTPS + HttpOnly + SameSite                        │

│  └── 检测：Session 指纹验证                                  │
│                                                                  │
│  CSRF：恶意页面诱导用户发起请求                              │
│  ├── 防护：CSRF Token                                        │
│  └── 防护：SameSite Cookie                                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```
