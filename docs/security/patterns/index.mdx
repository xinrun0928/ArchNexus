# 安全设计模式

安全设计模式是长期安全实践的结晶，是防御者思维的落地手段。

## 核心议题

安全不是功能，不能事后补救。防御者思维要求从设计阶段就考虑安全，最小权限原则要求只授予必要的权限，纵深防御确保一层被破还有他层，失效安全确保故障时不泄露信息。
