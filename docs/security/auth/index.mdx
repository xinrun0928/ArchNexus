# 认证与授权

认证与授权是系统安全的基础，决定了「你是谁」和「你能做什么」。

## 核心议题

认证（Authentication）和授权（Authorization）是两个不同的概念，必须正确区分。

**认证**回答的问题是「你是谁」，**授权**回答的是「你能做什么」。这两个问题必须分开处理，不能混为一谈。
