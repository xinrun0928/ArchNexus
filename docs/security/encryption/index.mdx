# 加密解密

加密解密是保护数据机密性的核心技术手段。

## 核心议题

加密算法分为对称加密和非对称加密，各有适用场景。哈希算法用于数据完整性验证和密码存储，不能用于加密解密。

**密钥管理**往往比算法本身更重要——密钥泄露等于安全失败。
