# 密钥管理

你的密钥直接写在代码里，上传到 GitHub 后整个系统被攻破。

问题出在哪？**没有正确的密钥管理**。

密钥管理是安全系统的基础，密钥泄露等于安全失败。

## AWS KMS 集成

```java
// AWS KMS 集成
@Service
public class AwsKmsService {
    private final AWSKMS kms;
    private final String keyId;

    public AwsKmsService() {
        this.kms = AWSKMSClientBuilder.defaultClient();
        this.keyId = System.getenv("AWS_KMS_KEY_ID");
    }

    // 加密
    public String encrypt(String plaintext) throws Exception {
        EncryptRequest request = new EncryptRequest()
            .withKeyId(keyId)
            .withPlaintext(SdkBytes.fromUtf8String(plaintext));

        EncryptResult result = kms.encrypt(request);
        return Base64.getEncoder().encodeToString(
            result.getCiphertextBlob().asByteArray());
    }

    // 解密
    public String decrypt(String ciphertext) throws Exception {
        byte[] ciphertextBytes = Base64.getDecoder().decode(ciphertext);

        DecryptRequest request = new DecryptRequest()
            .withKeyId(keyId)
            .withCiphertextBlob(SdkBytes.fromByteArray(ciphertextBytes));

        DecryptResult result = kms.decrypt(request);
        return result.getPlaintext().asUtf8String();
    }
}
```

```java
// Vault 集成
@Service
public class VaultKeyService {
    private final VaultTemplate vaultTemplate;

    public VaultKeyService() {
        VaultEndpoint endpoint = new VaultEndpoint();
        endpoint.fromUri("https://vault.example.com:8200");

        ClientAuthentication auth = new TokenAuthentication(
            System.getenv("VAULT_TOKEN"));

        VaultConfig config = new VaultConfig()
            .endpoint(endpoint)
            .authentication(auth)
            .build();

        this.vaultTemplate = new VaultTemplate(config);
    }

    // 读取密钥
    public String getSecret(String path) {
        VaultResponseSupport<Map<String, String>> response =
            vaultTemplate.read(path, Map.class);
        return response.getBody().get("password");
    }

    // 写入密钥
    public void putSecret(String path, String key, String value) {
        Map<String, Object> secrets = new HashMap<>();
        secrets.put(key, value);
        vaultTemplate.write(path, secrets);
    }
}
```

### 密钥轮换实现

```java
// 密钥版本管理
@Service
public class KeyVersionService {
    @Autowired
    private AwsKmsService kmsService;

    @Autowired
    private EncryptionKeyRepository keyRepository;

    // 获取当前密钥版本
    public EncryptionKey getCurrentKey() {
        return keyRepository.findByStatus(KeyStatus.ACTIVE)
            .stream()
            .max(Comparator.comparing(EncryptionKey::getVersion))
            .orElseThrow(() -> new KeyNotFoundException("没有可用密钥"));
    }

    // 轮换密钥
    @Transactional
    public EncryptionKey rotateKey() {
        // 1. 生成新密钥
        String keyId = UUID.randomUUID().toString();
        EncryptionKey newKey = new EncryptionKey();
        newKey.setKeyId(keyId);
        newKey.setStatus(KeyStatus.ACTIVE);
        newKey.setVersion(getCurrentKey().getVersion() + 1);
        newKey.setCreatedAt(LocalDateTime.now());

        // 2. 标记旧密钥为待废弃
        EncryptionKey currentKey = getCurrentKey();
        currentKey.setStatus(KeyStatus.PENDING_DEPRECATION);

        // 3. 保存新密钥
        keyRepository.save(currentKey);
        return keyRepository.save(newKey);
    }

    // 加密（使用最新密钥）
    public String encrypt(String plaintext) throws Exception {
        EncryptionKey currentKey = getCurrentKey();
        return kmsService.encrypt(plaintext) + "." + currentKey.getKeyId();
    }

    // 解密（根据密钥 ID 选择正确密钥）
    public String decrypt(String encrypted) throws Exception {
        String[] parts = encrypted.split("\\.");
        String ciphertext = parts[0];
        String keyId = parts[1];

        EncryptionKey key = keyRepository.findByKeyId(keyId)
            .orElseThrow(() -> new KeyNotFoundException("密钥不存在"));

        return kmsService.decrypt(ciphertext);
    }
}
```

## Spring Cloud Config 安全存储

```yaml
# bootstrap.yml - 从配置中心加密获取
spring:
  cloud:
    config:
      uri: https://config-server:8888
      username: user
      password: ${CONFIG_SERVER_PASSWORD}
```

```java
// 配置属性加密
@Configuration
@EnableConfigurationProperties(EncryptionProperties.class)
public class EncryptionConfig {

    @Bean
    public PropertySourceLocator propertySourceLocator(
            Environment environment,
            EncryptionService encryptionService) {

        return () -> {
            Map<String, Object> props = new HashMap<>();

            // 解密数据库密码
            String encryptedDbPassword =
                environment.getProperty("db.password.encrypted");
            if (encryptedDbPassword != null) {
                props.put("spring.datasource.password",
                    encryptionService.decrypt(encryptedDbPassword));
            }

            return new MapPropertySource("decrypted", props);
        };
    }
}
```

## 总结

密钥管理是安全系统的基础。

**存储方案**：

- 环境变量：简单场景
- KMS 服务：生产环境推荐
- Vault：企业级方案

**轮换策略**：

- 定期轮换
- 事件驱动轮换
- 兼容过渡

**最佳实践**：

- 不硬编码
- 不记录日志
- 分角色管理
- 审计访问
- 安全备份

**核心认知**：密钥管理是安全系统中最容易被忽视的环节，但也是最容易出问题的环节。再好的加密算法，如果密钥管理不当，都是徒劳。
