# 哈希算法

你的密码直接存数据库，被人脱库后用户密码全部泄露。

问题出在哪？**没有正确使用哈希算法存储密码**。

哈希算法是密码存储和数据完整性验证的基础。

## 哈希算法实现

```java
// 常用哈希算法实现
@Service
public class HashService {
    // SHA-256
    public String sha256(String input) throws NoSuchAlgorithmException {
        MessageDigest digest = MessageDigest.getInstance("SHA-256");
        byte[] hash = digest.digest(input.getBytes(StandardCharsets.UTF_8));
        return bytesToHex(hash);
    }

    // SHA-3 (Keccak)
    public String sha3(String input) throws NoSuchAlgorithmException {
        MessageDigest digest = MessageDigest.getInstance("SHA3-256");
        byte[] hash = digest.digest(input.getBytes(StandardCharsets.UTF_8));
        return bytesToHex(hash);
    }

    private String bytesToHex(byte[] bytes) {
        StringBuilder sb = new StringBuilder();
        for (byte b : bytes) {
            sb.append(String.format("%02x", b));
        }
        return sb.toString();
    }
}
```

## Bcrypt 实现

```java
// Bcrypt 密码哈希
@Service
public class PasswordHashService {
    private static final int BCRYPT_STRENGTH = 12;  // 计算强度

    // 生成密码哈希
    public String hashPassword(String plainPassword) {
        return BCrypt.hashpw(plainPassword, BCrypt.gensalt(BCRYPT_STRENGTH));
    }

    // 验证密码
    public boolean verifyPassword(String plainPassword, String hashedPassword) {
        return BCrypt.checkpw(plainPassword, hashedPassword);
    }
}

// Spring Security BCrypt
@Service
public class SpringPasswordService {
    @Autowired
    private PasswordEncoder passwordEncoder;

    // 加密密码
    public String encode(String rawPassword) {
        return passwordEncoder.encode(rawPassword);
    }

    // 验证密码
    public boolean matches(String rawPassword, String encodedPassword) {
        return passwordEncoder.matches(rawPassword, encodedPassword);
    }
}
```

## Argon2 实现

```java
// Argon2 密码哈希
@Service
public class Argon2PasswordService {
    private static final int MEMORY = 65536;   // 内存消耗
    private static final int ITERATIONS = 3;    // 迭代次数
    private static final int PARALLELISM = 4;   // 并行度

    // 生成 Argon2 哈希
    public String hashPassword(String plainPassword) {
        Argon2BytesGenerator generator = new Argon2BytesGenerator();
        Argon2Parameters params = new Argon2Parameters.Builder(Argon2Parameters.ARGON2id)
            .setMemoryCost(MEMORY)
            .setIterations(ITERATIONS)
            .setParallelism(PARALLELISM)
            .build();
        generator.init(params);

        byte[] hash = generator.generate(plainPassword.getBytes(StandardCharsets.UTF_8));
        return params.toString() + "$" + Base64.getEncoder().encodeToString(hash);
    }

    // 验证密码
    public boolean verifyPassword(String plainPassword, String storedHash) {
        try {
            Argon2BytesGenerator generator = new Argon2BytesGenerator();
            Argon2Parameters params = Argon2Parameters.fromEncoded(storedHash);
            generator.init(params);

            byte[] expectedHash = generator.generate(plainPassword.getBytes(StandardCharsets.UTF_8));
            byte[] actualHash = Base64.getDecoder().decode(
                storedHash.substring(storedHash.lastIndexOf('$') + 1));

            return MessageDigest.isEqual(expectedHash, actualHash);
        } catch (Exception e) {
            return false;
        }
    }
}
```

### HMAC 实现

```java
// HMAC 实现
@Service
public class HmacService {
    private static final String HMAC_ALGORITHM = "HmacSHA256";

    @Value("${hmac.secret}")
    private String secretKey;

    // 生成 HMAC
    public String generateMac(String message) throws Exception {
        Mac mac = Mac.getInstance(HMAC_ALGORITHM);
        SecretKey key = new SecretKeySpec(
            secretKey.getBytes(StandardCharsets.UTF_8), HMAC_ALGORITHM);
        mac.init(key);
        byte[] hmacBytes = mac.doFinal(message.getBytes(StandardCharsets.UTF_8));
        return bytesToHex(hmacBytes);
    }

    // 验证 HMAC
    public boolean verifyMac(String message, String mac) throws Exception {
        String expectedMac = generateMac(message);
        return MessageDigest.isEqual(
            expectedMac.getBytes(StandardCharsets.UTF_8),
            mac.getBytes(StandardCharsets.UTF_8));
    }
}
```

## 数据完整性验证

```java
// 文件完整性验证
@Service
public class IntegrityService {
    // 计算文件哈希
    public String calculateFileHash(File file) throws Exception {
        MessageDigest digest = MessageDigest.getInstance("SHA-256");

        try (FileInputStream fis = new FileInputStream(file)) {
            byte[] buffer = new byte[8192];
            int bytesRead;
            while ((bytesRead = fis.read(buffer)) != -1) {
                digest.update(buffer, 0, bytesRead);
            }
        }

        return bytesToHex(digest.digest());
    }

    // 验证文件完整性
    public boolean verifyIntegrity(File file, String expectedHash)
            throws Exception {
        String actualHash = calculateFileHash(file);
        return actualHash.equals(expectedHash);
    }

    // 计算并返回哈希（带盐）
    public String calculateHashWithSalt(String data, String salt) throws Exception {
        MessageDigest digest = MessageDigest.getInstance("SHA-256");
        digest.update(salt.getBytes(StandardCharsets.UTF_8));
        byte[] hash = digest.digest(data.getBytes(StandardCharsets.UTF_8));
        return bytesToHex(hash);
    }
}
```

## 总结

哈希算法是密码存储和数据完整性的基础。

**算法选择**：

- 密码存储：Bcrypt / Argon2（慢哈希）
- 数据完整性：SHA-256 / SHA-3（快哈希）
- 消息认证：HMAC-SHA256

**安全要点**：

- 密码必须加盐
- 使用慢哈希算法
- MD5 / SHA-1 已不安全

**核心认知**：哈希算法解决了「数据完整性验证」和「密码安全存储」问题，但不能单独解决加密解密问题。
