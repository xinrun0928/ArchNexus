# 对称加密

你的敏感数据直接存数据库，被人拖库后全部泄露。

问题出在哪？**没有对敏感数据进行加密存储**。

对称加密是最基础的加密方式。

## AES-CBC 实现

```java
// AES-CBC 加密
@Service
public class AesCbcEncryptionService {
    private static final String ALGORITHM = "AES/CBC/PKCS5Padding";
    private static final int KEY_SIZE = 256;  // 密钥长度
    private static final int IV_SIZE = 16;     // IV 长度

    @Value("${encryption.secret}")
    private String secretKey;  // Base64 编码的密钥

    // 生成随机 IV
    private byte[] generateIV() {
        byte[] iv = new byte[IV_SIZE];
        new SecureRandom().nextBytes(iv);
        return iv;
    }

    // 加密
    public String encrypt(String plaintext) throws Exception {
        // 1. 生成随机 IV
        byte[] iv = generateIV();

        // 2. 创建密钥
        Key key = new SecretKeySpec(
            Base64.getDecoder().decode(secretKey), "AES");

        // 3. 创建 Cipher
        Cipher cipher = Cipher.getInstance(ALGORITHM);
        cipher.init(Cipher.ENCRYPT_MODE, key,
            new IvParameterSpec(iv));

        // 4. 加密
        byte[] ciphertext = cipher.doFinal(plaintext.getBytes(StandardCharsets.UTF_8));

        // 5. 拼接 IV + 密文（IV 需要传递给解密方）
        byte[] combined = new byte[iv.length + ciphertext.length];
        System.arraycopy(iv, 0, combined, 0, iv.length);
        System.arraycopy(ciphertext, 0, combined, iv.length, ciphertext.length);

        return Base64.getEncoder().encodeToString(combined);
    }

    // 解密
    public String decrypt(String encrypted) throws Exception {
        byte[] combined = Base64.getDecoder().decode(encrypted);

        // 1. 分离 IV 和密文
        byte[] iv = new byte[IV_SIZE];
        byte[] ciphertext = new byte[combined.length - IV_SIZE];
        System.arraycopy(combined, 0, iv, 0, IV_SIZE);
        System.arraycopy(combined, IV_SIZE, ciphertext, 0, ciphertext.length);

        // 2. 创建密钥
        Key key = new SecretKeySpec(
            Base64.getDecoder().decode(secretKey), "AES");

        // 3. 创建 Cipher
        Cipher cipher = Cipher.getInstance(ALGORITHM);
        cipher.init(Cipher.DECRYPT_MODE, key,
            new IvParameterSpec(iv));

        // 4. 解密
        byte[] plaintext = cipher.doFinal(ciphertext);
        return new String(plaintext, StandardCharsets.UTF_8);
    }
}
```

## AES-GCM 实现（推荐）

```java
// AES-GCM 加密（带认证）
@Service
public class AesGcmEncryptionService {
    private static final String ALGORITHM = "AES/GCM/NoPadding";
    private static final int TAG_SIZE = 128;   // 认证标签大小
    private static final int IV_SIZE = 12;      // GCM 推荐 IV 大小

    @Value("${encryption.secret}")
    private String secretKey;

    // 加密
    public String encrypt(String plaintext) throws Exception {
        byte[] iv = new byte[IV_SIZE];
        new SecureRandom().nextBytes(iv);

        Key key = new SecretKeySpec(
            Base64.getDecoder().decode(secretKey), "AES");

        Cipher cipher = Cipher.getInstance(ALGORITHM);
        GCMParameterSpec parameterSpec = new GCMParameterSpec(TAG_SIZE, iv);
        cipher.init(Cipher.ENCRYPT_MODE, key, parameterSpec);

        byte[] ciphertext = cipher.doFinal(plaintext.getBytes(StandardCharsets.UTF_8));

        // 拼接 IV + 密文 + 认证标签
        byte[] combined = new byte[iv.length + ciphertext.length];
        System.arraycopy(iv, 0, combined, 0, iv.length);
        System.arraycopy(ciphertext, 0, combined, iv.length, ciphertext.length);

        return Base64.getEncoder().encodeToString(combined);
    }

    // 解密（自动验证完整性）
    public String decrypt(String encrypted) throws Exception {
        byte[] combined = Base64.getDecoder().decode(encrypted);

        byte[] iv = new byte[IV_SIZE];
        byte[] ciphertext = new byte[combined.length - IV_SIZE];
        System.arraycopy(combined, 0, iv, 0, IV_SIZE);
        System.arraycopy(combined, IV_SIZE, ciphertext, 0, ciphertext.length);

        Key key = new SecretKeySpec(
            Base64.getDecoder().decode(secretKey), "AES");

        Cipher cipher = Cipher.getInstance(ALGORITHM);
        GCMParameterSpec parameterSpec = new GCMParameterSpec(TAG_SIZE, iv);
        cipher.init(Cipher.DECRYPT_MODE, key, parameterSpec);

        byte[] plaintext = cipher.doFinal(ciphertext);
        return new String(plaintext, StandardCharsets.UTF_8);
    }
}
```

## 密钥生成

```java
// 密钥生成
public class KeyGenerator {
    public static void main(String[] args) throws Exception {
        // 生成 AES-256 密钥
        KeyGenerator keyGen = KeyGenerator.getInstance("AES");
        keyGen.init(256);
        SecretKey key = keyGen.generateKey();

        // 输出 Base64 编码的密钥
        String base64Key = Base64.getEncoder().encodeToString(key.getEncoded());
        System.out.println("AES-256 Key: " + base64Key);

        // 生成随机盐（用于密码处理）
        SecureRandom random = new SecureRandom();
        byte[] salt = new byte[16];
        random.nextBytes(salt);
        String base64Salt = Base64.getEncoder().encodeToString(salt);
        System.out.println("Salt: " + base64Salt);
    }
}
```

## 总结

对称加密是最基础的加密方式。

**推荐算法**：

- AES-256-GCM：安全可靠，推荐使用
- AES-256-CBC：可用，但不如 GCM

**安全要点**：

- 密钥随机生成，长度足够
- 每次加密使用新的随机 IV
- 密钥安全存储，不硬编码
- 优先使用 GCM 模式（带认证）

**核心认知**：对称加密解决了「数据保密性」问题，但密钥本身的安全管理才是最大的挑战。
