## 学习路径

建议按以下顺序学习：

1. **认证与授权**：理解身份认证和权限控制的基础概念
2. **加密解密**：掌握数据保护的核心技术
3. **会话管理**：保护用户身份不被冒用
4. **安全设计模式**：建立正确的安全思维方式
5. **防御深度**：理解多层安全防护体系
6. **OAuth 与 OpenID**：掌握现代身份认证协议
