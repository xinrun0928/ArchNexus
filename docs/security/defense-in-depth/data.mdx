## 敏感数据加密

```java
// 敏感数据加密服务
@Service
public class SensitiveDataEncryption {

    @Autowired
    private EncryptionService encryptionService;

    // 用户密码：BCrypt 哈希
    public void setUserPassword(User user, String rawPassword) {
        user.setPassword(bcryptEncoder.encode(rawPassword));
    }

    // 身份证号：AES 加密
    public void setIdCard(User user, String idCard) {
        if (idCard != null) {
            String encrypted = encryptionService.encrypt(idCard);
            user.setIdCardEncrypted(encrypted);
            // 明文不存储
            user.setIdCard(null);
        }
    }

    // 银行卡号：AES 加密
    public void setBankCard(User user, String bankCard) {
        if (bankCard != null) {
            String encrypted = encryptionService.encrypt(bankCard);
            user.setBankCardEncrypted(encrypted);
            user.setBankCard(null);
        }
    }

    // 医疗记录：AES 加密
    public void setMedicalRecord(Patient patient, MedicalRecord record) {
        String encrypted = encryptionService.encrypt(
            objectMapper.writeValueAsString(record)
        );
        patient.setMedicalRecordEncrypted(encrypted);
    }
}

// 数据库字段加密配置
@Entity
public class User {
    // 密码：BCrypt
    @Column(nullable = false)
    private String password;

    // 身份证号：AES 加密
    @Column
    @Convert(converter = AesEncryptionConverter.class)
    private String idCardEncrypted;

    // 手机号：AES 加密
    @Column
    @Convert(converter = AesEncryptionConverter.class)
    private String phoneEncrypted;

    // 邮箱：可搜索加密
    @Column
    @Convert(converter = SearchableEncryptionConverter.class)
    private String emailEncrypted;
}

// 加密转换器
@Converter
public class AesEncryptionConverter implements AttributeConverter<String, String> {

    @Autowired
    private transient EncryptionService encryptionService;

    @Override
    public String convertToDatabaseColumn(String attribute) {
        if (attribute == null) {
            return null;
        }
        return encryptionService.encrypt(attribute);
    }

    @Override
    public String convertToEntityAttribute(String dbData) {
        if (dbData == null) {
            return null;
        }
        return encryptionService.decrypt(dbData);
    }
}
```

## 数据库安全配置

```sql
-- 数据库安全配置

-- 创建应用专用账号（最小权限）
CREATE USER 'app_user'@'%' IDENTIFIED BY 'strong_password';
GRANT SELECT, INSERT, UPDATE ON production.users TO 'app_user'@'%';
GRANT SELECT, INSERT, UPDATE ON production.orders TO 'app_user'@'%';
GRANT SELECT ON production.products TO 'app_user'@'%';
FLUSH PRIVILEGES;

-- 审计日志表
CREATE TABLE audit_log (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    user_id BIGINT,
    action VARCHAR(100),
    resource_type VARCHAR(100),
    resource_id VARCHAR(100),
    ip_address VARCHAR(50),
    user_agent VARCHAR(500),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_user_id (user_id),
    INDEX idx_created_at (created_at)
);

-- 敏感操作触发器
DELIMITER //

CREATE TRIGGER before_user_update_audit
BEFORE UPDATE ON users
FOR EACH ROW
BEGIN
    -- 记录密码变更
    IF OLD.password != NEW.password THEN
        INSERT INTO audit_log (user_id, action, resource_type, resource_id)
        VALUES (NEW.id, 'PASSWORD_CHANGE', 'USER', NEW.id);
    END IF;

    -- 记录敏感字段变更
    IF OLD.id_card_encrypted != NEW.id_card_encrypted THEN
        INSERT INTO audit_log (user_id, action, resource_type, resource_id)
        VALUES (NEW.id, 'SENSITIVE_FIELD_CHANGE', 'USER', NEW.id);
    END IF;
END //

DELIMITER ;
```

## 数据脱敏

```java
// 数据脱敏工具
@Service
public class DataMaskingService {

    // 手机号脱敏：138****5678
    public String maskPhone(String phone) {
        if (phone == null || phone.length() < 7) {
            return phone;
        }
        return phone.substring(0, 3) + "****" +
               phone.substring(phone.length() - 4);
    }

    // 身份证脱敏：310***********1234
    public String maskIdCard(String idCard) {
        if (idCard == null || idCard.length() < 10) {
            return idCard;
        }
        return idCard.substring(0, 3) + "***********" +
               idCard.substring(idCard.length() - 4);
    }

    // 银行卡脱敏：**** **** **** 1234
    public String maskBankCard(String bankCard) {
        if (bankCard == null || bankCard.length() < 4) {
            return bankCard;
        }
        return "**** **** **** " + bankCard.substring(bankCard.length() - 4);
    }

    // 邮箱脱敏：t***@example.com
    public String maskEmail(String email) {
        if (email == null || !email.contains("@")) {
            return email;
        }
        String[] parts = email.split("@");
        if (parts[0].length() <= 1) {
            return email;
        }
        return parts[0].charAt(0) + "***@" + parts[1];
    }

    // 姓名脱敏：张*、欧阳*海
    public String maskName(String name) {
        if (name == null || name.isEmpty()) {
            return name;
        }
        if (name.length() == 2) {
            return name.charAt(0) + "*";
        }
        return name.substring(0, name.length() - 1) + "*";
    }
}

// 脱敏注解
@Target({ElementType.FIELD, ElementType.METHOD})
@Retention(RetentionPolicy.RUNTIME)
public @interface Masked {
    MaskType type() default MaskType.PHONE;
}

public enum MaskType {
    PHONE, ID_CARD, BANK_CARD, EMAIL, NAME, CUSTOM
}
```

## 数据备份

```bash
#!/bin/bash
# MySQL 数据备份脚本

BACKUP_DIR="/backup/mysql"
DATE=$(date +%Y%m%d_%H%M%S)
RETENTION_DAYS=30

# 创建备份目录
mkdir -p ${BACKUP_DIR}

# 全量备份
mysqldump -h ${DB_HOST} -u ${DB_USER} -p${DB_PASSWORD} \
    --single-transaction \
    --routines \
    --triggers \
    --events \
    --master-data=2 \
    --flush-logs \
    production | gzip > ${BACKUP_DIR}/full_backup_${DATE}.sql.gz

# 计算校验和
sha256sum ${BACKUP_DIR}/full_backup_${DATE}.sql.gz > \
    ${BACKUP_DIR}/full_backup_${DATE}.sql.gz.sha256

# 删除过期备份
find ${BACKUP_DIR} -name "*.sql.gz" -mtime +${RETENTION_DAYS} -delete

# 上传到对象存储
aws s3 cp ${BACKUP_DIR}/full_backup_${DATE}.sql.gz \
    s3://${S3_BUCKET}/mysql-backups/ \
    --sse AES256

echo "Backup completed: ${DATE}"
```

## 数据恢复演练

```bash
#!/bin/bash
# 数据恢复演练脚本

BACKUP_DATE=$1
ENVIRONMENT="staging"
BACKUP_DIR="/backup/mysql"

if [ -z "$BACKUP_DATE" ]; then
    echo "Usage: $0 <YYYYMMDD_HHMMSS>"
    exit 1
fi

BACKUP_FILE="${BACKUP_DIR}/full_backup_${BACKUP_DATE}.sql.gz"

# 检查备份文件是否存在
if [ ! -f "$BACKUP_FILE" ]; then
    echo "Backup file not found: $BACKUP_FILE"
    exit 1
fi

# 验证校验和
echo "Verifying backup integrity..."
sha256sum -c "${BACKUP_FILE}.sha256" || {
    echo "Checksum verification failed!"
    exit 1
}

# 恢复到测试环境
echo "Restoring to staging environment..."
gunzip < "$BACKUP_FILE" | \
    mysql -h staging-mysql -u restore_user -p restore_password staging

# 验证数据
echo "Verifying restored data..."
mysql -h staging-mysql -u verify_user -p verify_password staging \
    -e "SELECT COUNT(*) as user_count FROM users;"

echo "Recovery drill completed successfully"
```

## 总结

数据是企业最重要的资产，必须重点保护。

**数据分类**：

- 公开数据：无特殊保护
- 内部数据：访问控制
- 敏感数据：加密存储
- 机密数据：最高级别保护

**防护措施**：

- 敏感数据加密：AES、BCrypt
- 数据库访问控制：最小权限
- 数据脱敏：展示时隐藏敏感信息
- 数据备份：定期备份 + 异地存储
- 恢复演练：定期测试恢复能力

**合规要求**：

- GDPR / PIPL：隐私保护
- PCI-DSS：支付数据保护
- 数据泄露通知

**核心认知**：数据泄露的后果是灾难性的。加密存储、严格访问控制、定期备份、安全审计是数据安全的基本要求。
