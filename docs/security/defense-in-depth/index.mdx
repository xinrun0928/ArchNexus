# 防御深度

防御深度强调在网络的多个层面、应用的多层架构中部署安全措施。

## 核心议题

安全没有银弹。网络层、应用层、数据层、身份认证层、监控响应层，每一层都要有独立的安全能力。网络层防护是第一道屏障，数据层防护是最后底线。

## 内容导览

| 文章 | 核心内容 |
|---|---|
| [网络层防护](./network) | 防火墙、VPC、DDoS 防护 |
| [应用层防护](./application) | SQL/XSS 注入，越权访问 |
| [数据层防护](./data) | 加密存储，备份恢复 |
| [安全监控与审计](./monitoring) | 日志，SIEM，应急响应 |
