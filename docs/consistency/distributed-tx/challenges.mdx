# 分布式事务的挑战

你的系统跨多个数据库或服务，但事务不生效了。

问题出在哪？**没有理解分布式事务的挑战**。

分布式事务是分布式系统中最复杂的问题之一。

## 总结

分布式事务面临多重挑战。

**核心挑战**：

- 网络不可靠
- 节点可能故障
- 部分失败问题
- CAP 约束

**解决思路**：

- 强一致性方案：2PC、3PC
- 最终一致性方案：TCC、本地消息表、Saga
- 业务补偿：最大努力通知

**选择原则**：

- 强一致性：金融、订单等关键数据
- 最终一致性：非关键数据、高并发场景
- 业务补偿：可以容忍部分失败的场景
